## 模块一：公司概况与行业定位

### 1.1 公司基本信息

重庆长安汽车股份有限公司（以下简称"长安汽车"或"公司"）是中国四大汽车集团之一，总部位于重庆。公司主营业务涵盖乘用车、商用车、新能源汽车及核心零部件的研发、生产与销售。

- **成立时间**：1996年
- **上市时间**：1997年（深交所主板）
- **实际控制人**：中国兵器装备集团有限公司（央企背景）
- **主要品牌**：长安、深蓝、阿维塔、启源等
- **2025年销量**：约268万辆（需从年报确认精确值）

### 1.2 行业定位

| 维度 | 说明 |
|------|------|
| 申万一级行业 | 汽车 |
| 申万二级行业 | 汽车整车 |
| 行业地位 | 中国汽车行业头部企业，自主品牌销量领先 |
| 竞争格局 | 与比亚迪、上汽、长城、广汽、吉利等形成多强竞争 |
| 行业趋势 | 新能源转型加速，智能化竞争白热化，价格战持续 |

### 1.3 2025年行业环境

2025年中国汽车行业面临以下关键环境变化：

- **价格战持续**：新能源汽车渗透率突破50%，传统燃油车市场加速萎缩，行业整体利润率承压
- **新能源转型**：公司深蓝、阿维塔等新能源品牌持续放量，但盈利贡献仍需时间
- **智能化竞争**：高阶智驾成为差异化竞争核心，研发投入持续加大
- **出口增长**：中国汽车出口保持增长势头，海外市场成为新增长点

## 模块十一：风险识别与评估

### 11.1 核心风险链条

**风险链条一：价格战→盈利下滑→现金流恶化**

行业价格战持续 → 整车降价促销 → 毛利率下滑 → 净利率从7.49%降至2.48% → 归母净利润从113亿降至41亿 → 经营现金流从199亿降至18亿 → 自由现金流为负 → 依赖融资维持投资

**风险链条二：新能源转型→投入加大→短期亏损**

新能源渗透率提升 → 燃油车市场萎缩 → 公司加速新能源转型 → 深蓝、阿维塔等品牌投入加大 → 研发费用、销售费用上升 → 新能源品牌尚未盈利 → 拖累整体业绩

**风险链条三：应收票据→兑付风险→流动性压力**

应收票据329.98亿元规模较大 → 若票据到期无法兑付 → 流动性收紧 → 短期偿债压力上升
