# DGX Spark Hackathon 参赛材料

本目录集中保存 SIQ Research Engine 的公开赛事说明。团队联系方式、成员个人
信息、合影原图、视频后台信息和组委会表单内容不放入公开代码仓库。

## 阅读顺序

1. [项目提交说明](./dgx-spark-hackathon-submission-package.md)：600 字以上项目
   说明、技术架构、DGX Spark 部署、模型优化、技术栈和演示方案。
2. [评审对照与演示指南](./dgx-spark-hackathon-evaluation-guide.md)：将项目能力
   映射到比赛评分标准，并给出演示与“十日谈”证据组织方式。
3. 根级 [`README.md`](../../README.md)：完整产品、架构、功能和运行说明。
4. [`LICENSE`](../../LICENSE)、[`NOTICE`](../../NOTICE) 与
   [`THIRD_PARTY_LICENSES.md`](../../THIRD_PARTY_LICENSES.md)：开源及第三方许可
   边界。

## 公开边界

- 代码仓库：<https://github.com/maoyadongsh/siq-research-engine>
- 公开文档不预填团队隐私、联系方式、合影和未发布视频 URL。
- 本目录不保存模型权重、客户数据、真实凭据或未经脱敏的运行证据。
- 文档中的动态服务状态必须在正式演示和提交前重新验证。
- 架构图当前仅在本地归档，尚未作为公开赛事材料提交。
