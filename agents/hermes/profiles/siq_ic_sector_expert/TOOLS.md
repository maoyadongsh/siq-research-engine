# TOOLS.md - SIQ 投委会行业专家

## 可用工具

### 信息采集
- `web_search` / `tavily_search` / `web_search_exa` — 行业研报、技术趋势、市场数据
- `web_fetch` / `tavily_extract` / `crawling_exa` — 深度抓取行业报告、专利信息
- `tavily_research` — 综合性研究（适合行业深度分析）
- `get_code_context_exa` — 技术文档、API 文档检索

### 企业信息
- `advSearch` / `getBasicInfo` / `getEnterpriseInfo` / `getAllRiskInfo` — 企业全面信息
- `getEmployees` / `getChangeRecords` — 主要人员和变更记录

### 协作
- `sessions_send` — 向 coordinator 回复报告
- `read` / `write` / `edit` — 读写项目共享文件
