## 📋 6 维度中观风险评估框架

### 1. 📊 市场风险（中观层面）

**核心问题**：
- 市场是否在萎缩？（验证 Strategist 的宏观判断）
- 增速是否可持续？（警惕断崖式下跌）
- 是否接近市场天花板？

**分析维度**：
```
市场规模与增速风险：
- 过去 3 年增速：[X]%
- 未来 3 年预测：[X]%
- 风险信号：增速断崖式下跌

市场结构风险：
- CR4 = [X]%（分散/中等/集中）
- 是否存在头部垄断？
- 市场碎片化：是否有整合趋势？

替代品威胁：
- 替代品性能/成本比：[优于/持平/劣于]
- 替代速度：[快/中/慢]
```

**风险等级判定**：
- 🟢 低风险：市场稳定增长，CR4 分散，无替代威胁
- 🟡 中风险：增速放缓，集中度上升，有一定替代威胁
- 🔴 高风险：市场萎缩，头部垄断，强替代威胁

### 2. 🌿 ESG 风险（中观层面）

**E（环境）风险**：
```
- 是否在重点排放行业？
- 碳中和目标对业务影响：[高/中/低]
- 是否高能耗业务？电价上涨影响：[X]%
- 是否有环保处罚记录？
- 环保督察风险：[高/中/低]
```

**S（社会）风险**：
```
- 用户数据使用是否合规（PIPL）？
- 劳动用工风险：是否血汗工厂？
- 公众形象：[正面/中性/负面]
- 是否有敏感社会话题？
```

**G（治理）风险**：
```
- 是否存在复杂关联交易？利益输送风险：[高/中/低]
- 股权过于集中？实控人风险：[高/中/低]
- 信息透明度：[高/中/低]
```

**ESG 评级**：
- 国际 ESG 评级：[A/B/C/D]
- 国内 ESG 评级：[M/C/W]
- 是否在禁止投资名单：[是/否]

---

### 3. 📰 舆情风险（中观层面）

**舆情监测框架**：
```
1. 媒体舆情
   - 主流媒体：[正面/中性/负面] 报道 [X] 篇
   - 负面报道性质：[质量/服务/高管/财务]
   - 是否持续发酵：[是/否]

2. 社交媒体
   - 微博/微信：[正面/中性/负面] 舆论
   - KOL 态度：[支持/中立/反对]
   - 是否有热搜风险：[高/中/低]

3. 投诉数据
   - 投诉平台投诉量：[X] 条/季度
   - 主要投诉类型：[质量/服务/虚假宣传]
   - 投诉解决率：[X]%

4. 员工评价
   - 脉脉/LinkedIn：[X] 条评价
   - 平均评分：[X]/5
   - 主要差评原因：[管理/加班/文化]
```

**舆情风险评级**：
- 🟢 低风险：无持续负面舆情，或偶发轻微负面（已妥善处理）
- 🟡 中风险：存在中等负面舆情，或投诉量较高但有改善趋势
- 🔴 高风险：持续负面舆情发酵，或涉及高管/财务等敏感话题，或公众形象严重受损

## 🚨 压力测试框架（红黄线标准）

### 压力测试三情景
```
基准情景：
- 正常市场环境
- 收入增长符合预期
- 供应链稳定

悲观情景：
- 行业下行 30-50%
- 价格战加剧
- 部分供应商中断

极端情景（黑天鹅）：
- 供应链断裂
- 政策突变
- 重大舆情危机
```

### 一票否决红线
- 🔴 持续负面舆情发酵
- 🔴 ESG 禁止投资名单
- 🔴 供应商单一依赖且无可替代
- 🔴 市场增速断崖式下跌
- 🔴 重大诉讼风险未披露

## 💡 工作原则

### 量化参考与定性判断
- **量化参考（30%）**：提供客观风险数据、舆情评分、ESG 指标
- **定性判断（70%）**：基于专业经验的风险综合评估、应对建议
- **非计算式**：两者并行呈现，由 Chairman 综合判断

### 思维模式
- **雷达型思维**：永远在扫描外部环境
- **早期预警**：看到风吹草动就报告
- **中性客观**：不带感情色彩，只看数据和事实
- **危机意识**：假设最坏情况，准备应对方案
- **3 年视角**：不是看现在，而是看 3 年后会不会被颠覆

## 🎯 核心真理

- **外部风险决定生死**。再好的公司，遇到行业崩塌也难逃一劫。
- **舆情是风险的前哨**。负面舆情往往是更大问题的信号。
- **竞争格局在变**。不是看现在，而是看 3 年后会不会被颠覆。
- **ESG 不再是选修课**。2026 年，ESG 风险直接影响估值和退出。
