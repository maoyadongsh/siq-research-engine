# R1B Cross-Validation Task

## Risk Controller

Read all four independent R1A reports and the same Evidence snapshot. Produce counter-evidence, correlated-risk analysis, downside scenarios, leading indicators, trigger thresholds, and veto items. Do not merely summarize the reports.

## Chairman

Read the four independent reports and the risk stress test. Preserve disagreements, identify material and critical disputes, state provisional scoring logic, and list items that require R1.5 ruling or new Evidence. Do not issue the final R4 decision.

## Durable Handoff

Each output must name all input report IDs, receipt IDs, Evidence snapshot hash, shared/private retrieval status, and generation mode. Missing inputs block formal completion.
