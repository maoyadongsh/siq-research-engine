- 【方括号】内容需根据项目实际情况填写
- 专家评分需提供简要理由（1-2 句话）
- 主席六维权重按项目阶段自动切换（参见 `agents/hermes/profiles/siq_ic_shared/ic_workflow_policy.json → chairman_scoring`）
- V2 加权汇总：chairman 30% + strategy/sector/finance/risk 各 15% + legal 10%
- 阈值：≥70 通过 / 68-69 复议 / <70 不通过

## 附录 B：评分标准参考

### 专家评分（0-100）

| 分数区间 | 评级 | 含义 |
|---------|------|------|
| 81-100 | ⭐⭐⭐ 强烈推荐 | 基本面优异，风险极低 |
| 71-80 | ⭐⭐ 积极 | 基本面健康，风险可控 |
| 61-70 | ⭐ 中性偏谨慎 | 有亮点但存在显著风险 |
| 41-60 | ⚠️ 偏负面 | 风险显著大于机会 |
| 0-40 | 🚫 严重负面 | 存在红旗/硬伤 |

### 主席六维评分（0-10）

| 得分 | 等级 | 定义 |
|------|------|------|
| 9.0-10 | 卓越 | 行业顶尖水平 |
| 8.0-8.9 | 优秀 | 行业领先水平 |
| 7.0-7.9 | 良好 | 行业平均以上 |
| 6.0-6.9 | 可接受 | 达到基本标准 |
| 5.0-5.9 | 需谨慎 | 低于行业标准 |
| <5.0 | 不合格 | 存在重大缺陷 |

### 评分纪律

- 置信度 Low 时，分数浮动范围 ±15
- 存在红旗项，评分不得超过 60（除非有充分对冲方案）
- 未验证假设超过 3 项，评分不得超过 70
- 数据来源必须标注（verified / assumed / estimated）
