# 投资者5分钟速读模板

本模板应置于执行摘要之前，作为报告的"通俗版总结"。

## 结构要求

### 1. 💰 这家公司是做什么的？（一句话）
- 必须包含：主营业务 + 核心产品 + 行业地位
- 使用通俗语言，避免专业术语
- 示例："国内领先的新能源汽车制造商，主要生产电动乘用车和动力电池"

### 2. 📈 今年表现如何？（改善/恶化/稳定）
使用直观的状态标识：
- ✅ 显著改善：核心指标全面向好
- ⚠️ 结构性分化：有改善有恶化
- ❌ 明显恶化：多项核心指标下滑
- ➡️ 基本稳定：与去年持平

关键指标速览：
- 营收：XXX亿元（↑/↓ X%）
- 净利润：XXX亿元（↑/↓ X%）
- 经营现金流：XXX亿元（↑/↓ X%）

用一句话总结：例如"营收增长但盈利下滑，现金流承压"

### 3. 💵 赚的钱质量高吗？（现金流含金量）
用通俗语言解释：
- 🟢 质量优秀：赚的每1块钱利润，收到了X元现金（>1.0）
- 🟡 质量一般：赚的钱部分变成了应收账款（0.7-1.0）
- 🔴 质量较差：账面赚钱，但没收到现金（<0.7）

关键问题：
- 客户付款及时吗？（应收账款天数）
- 存货能卖出去吗？（存货周转天数）
- 现金够用吗？（现金/短期债务）

### 4. ⚠️ 主要风险是什么？（3-5个关键风险）
用"如果...那么..."句式：
- 🔴 高风险：如果短期借款无法续接，可能面临流动性危机
- 🟡 中风险：如果毛利率继续下滑，盈利能力将承压
- 🟢 低风险/观察项：需关注关联交易是否公允

风险分类：
- 💰 财务风险：债务、现金流、盈利
- 🏭 经营风险：竞争、客户、供应链
- ⚖️ 合规风险：诉讼、处罚、治理
- 🌍 外部风险：政策、周期、不可抗力

### 5. 🎯 值得关注的改善信号（跟踪指标）
什么情况下会变好？
- ✅ 毛利率回升到X%以上
- ✅ 应收账款回收率超过X%
- ✅ 新产品收入占比提升到X%

什么情况下要警惕？
- ❌ 经营现金流连续X个季度为负
- ❌ 短期债务/货币资金比率超过X
- ❌ 应收账款坏账准备大幅增加

下次关注时点：
- 📅 季报披露（X月X日）
- 📅 年度股东大会（X月X日）
- 📅 重大公告跟踪

## 语言风格要求

### 避免使用的专业术语
| ❌ 不要这样说 | ✅ 应该这样说 |
|------------|------------|
| 应收账款周转率下降 | 客户付款变慢了 |
| 毛利率压缩 | 每卖一块钱赚的更少了 |
| 自由现金流为负 | 赚的钱不够公司发展投入 |
| 资产负债率上升 | 借的钱占比增加了 |
| ROIC下滑 | 投资回报率降低 |
| 存货减值 | 库存商品卖不出去，贬值了 |

### 推荐使用的通俗表达
- "账上有钱" 代替 "货币资金充裕"
- "赚钱能力" 代替 "盈利能力"
- "收到现金" 代替 "销售回款"
- "借钱" 代替 "融资"
- "竞争对手" 代替 "同业企业"

## 输出示例

```markdown
# 📱 投资者5分钟速读

## 💰 这家公司是做什么的？
国内领先的新能源汽车制造商，主要生产电动乘用车、混合动力车和动力电池，市场份额位居行业前三。

## 📈 今年表现如何？
⚠️ **结构性分化** - 营收大幅增长，但盈利能力下滑

**关键数据**：
- 营收：1,530亿元（↑42%）
- 净利润：125亿元（↓15%）
- 经营现金流：85亿元（↓35%）

**一句话总结**：销量快速增长，但价格战导致毛利率下降，现金回收变慢。

## 💵 赚的钱质量高吗？
🟡 **质量一般** - 账面赚了125亿利润，但只收到85亿现金

**关键问题**：
- ❌ 客户付款变慢：应收账款从60天增加到90天
- ⚠️ 存货增加：存货周转从45天增加到65天
- ✅ 现金还够用：账上有350亿，短期债务200亿

**通俗解释**：公司卖车卖得很快，但客户（主要是经销商）付款慢了，库存也积压了一些。好在账上现金充足，短期没有资金压力。

## ⚠️ 主要风险是什么？

### 🔴 高风险
1. **价格战持续**：如果行业价格战持续，毛利率可能进一步下滑至12%以下，威胁盈利能力

### 🟡 中风险
2. **应收账款风险**：如果经销商资金链出现问题，可能产生坏账损失
3. **原材料价格波动**：如果锂电池原材料价格上涨20%，成本压力加大

### 🟢 观察项
4. **技术路线风险**：需关注固态电池等新技术对当前产品线的冲击
5. **补贴政策变化**：需关注新能源补贴政策调整影响

## 🎯 值得关注的改善信号

### ✅ 什么情况下会变好？
- 毛利率回升到18%以上（当前15%）
- 应收账款回收加速，周转天数降到75天以内
- 新车型上市，单车售价提升10%+

### ❌ 什么情况下要警惕？
- 经营现金流连续2个季度为负
- 毛利率跌破12%
- 应收账款坏账准备大幅增加（目前3%）

### 📅 下次关注时点
- **Q2季报**（7月31日前）：关注价格战是否缓和
- **半年度业绩说明会**（8月）：管理层战略调整
- **重大公告**：新车型上市、价格调整
```

## 质量检查清单

生成速读版时必须确保：
- [ ] 避免所有专业术语，或在首次出现时用通俗语言解释
- [ ] 使用具体数字而非百分比（"85亿" 而非 "下降35%"）
- [ ] 使用视觉符号（✅❌⚠️）增强可读性
- [ ] 每个风险都有"如果...那么..."的具体描述
- [ ] 改善信号都有明确的数字阈值
- [ ] 提供跳转链接方便深入阅读

## 注意事项

1. **不要过度简化**：通俗≠不准确，核心数字必须准确
2. **保持中立**：不做买入/卖出建议，只陈述事实
3. **突出重点**：5分钟速读必须控制在800-1200字
4. **视觉友好**：多用表格、符号、分点，少用长段落
