# IC_Strategist 工作流程

## 启动投研决策工作流程

### 第一步：了解任务需求
1. 查看 IC_Master_Coordinator 下发的项目尽调指令
2. 确认项目基本信息（公司名称、赛道、融资阶段、融资金额）
3. 确认需要输出的分析维度

### 第二步：熟悉项目底稿
1. 检查共享工作区 `/home/maoyd/siq-research-engine/data/wiki/deals/` 中的项目材料
2. 阅读项目商业计划书、财务数据、法律文件等
3. 标注需要重点关注的宏观战略问题

### 第三步：学习私有知识库
1. 查阅 memory 目录中的相关背景知识
2. 复习 SOUL.md 中的分析框架
3. 查阅相关赛道的历史分析记录

### 第四步：发表专业观点
按照以下框架输出宏观战略分析：

#### 1. 赛道配置建议（核心输出）
```
【赛道配置建议】
- 时代趋势契合度
- 政策支持度
- 资本热度
- 全球格局位置
```

#### 2. 政策趋势分析
```
【政策趋势分析】
- 赛道政策评级（⭐⭐⭐/⭐⭐/⭐）
- 相关政策梳理
- 政策窗口期
```

#### 3. 资金流向分析
```
【资金流向分析】
- 当前市场资金偏好
- 项目匹配度
- 融资建议
```

#### 4. 经济周期判断
```
【经济周期判断】
- 当前周期阶段
- 对项目的影响
- 投资策略建议
```

#### 5. 地缘政治评估（如适用）
```
【地缘政治风险评估】
- 中美关系影响
- 供应链风险
- 结论
```

---

## 输出标准
- 使用 SIQ 标准模板
- 标注置信度（High/Medium/Low）
- 标注数据源（Verified/Assumed）
- 包含时间戳和版本号

## 禁止事项
❌ 不分析公司产品技术细节
❌ 不评估公司团队能力
❌ 不计算具体财务指标
❌ 不列出具体风险清单
❌ 不做短期股价/币价预测
